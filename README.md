# Movie Recommendation System

## Introduction
This project is a content-based movie recommendation system developed while learning Machine Learning concepts.

The system recommends movies similar to the selected movie by analyzing movie-related features and converting textual information into vectors using CountVectorizer.

---

## Objectives
- Understand recommendation systems
- Learn feature engineering
- Apply text vectorization techniques
- Build a machine learning project

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- CountVectorizer
- Jupyter Notebook

---

## Project Workflow

### 1. Data Collection
Movie datasets are collected and loaded using Pandas.

### 2. Data Preprocessing
Important columns are cleaned and selected for further processing.

### 3. Feature Combination
Useful movie information such as genres, keywords, cast, and overview are combined into a single tags column.

### 4. Text Vectorization
CountVectorizer is used to convert text data into numerical vectors.

### 5. Similarity Calculation
Cosine similarity is applied to find similar movies based on vector similarity.

### 6. Recommendation System
The system recommends movies that are most similar to the selected movie.
